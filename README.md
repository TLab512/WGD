# Wavelet-Guided Diffusion with Geometric Modulation for Single-View 3D Reconstruction


This repository provides the official implementation of WGDM. The current release includes the core network architecture for the review process of *The Visual Computer*. The full training and evaluation pipeline will be made publicly available after the completion of the review process. The complete training and evaluation code will be publicly released after the review process. The public datasets used in this work are available at [ShapeNet](https://shapenet.org/) and [Pix3D](https://github.com/xingyuansun/pix3d).
