# WGD: Wavelet-Enhanced Geometry-Aware Diffusion for Single-View 3D Point Cloud Reconstruction


This repository provides the official implementation of WGD. The current release contains the core network architecture. The complete training and evaluation code will be publicly released after the review process. The public datasets used in this work are available at [ShapeNet](https://shapenet.org/) and [Pix3D](https://github.com/xingyuansun/pix3d).
